# amr-geekplus adapter

AMR Geek+ device adapter — transport: REST + WebSocket (RCS fleet manager).

- **Language:** Go
- **Port:** 9093
- **Run:** `cd services/adapters/amr-geekplus && go run .`
- **Health:** `GET http://localhost:9093/healthz`

Implements the uniform internal device contract (see [build.md](../../../build.md) §8).
